# Daily Logs Hub

Daily logs are the dashboard for the whole day. Use them to quickly capture what happened, then link to the detailed session notes in each training area.

```button
name Today Example
type link
action obsidian://open?vault=Personal%20Fitness&file=00%20Daily%20Logs%2F2026%2F05%20May%2F2026-05-14
color blue
```

```button
name Daily Dashboard
type link
action obsidian://open?vault=Personal%20Fitness&file=05%20Dashboards%2FDaily%20Log%20Dashboard
color blue
```

## Naming
Use:

```text
YYYY-MM-DD.md
```

Example:

```text
2026-05-14.md
```

## Folder Pattern
Put each daily note in the matching year and month folder:

```text
00 Daily Logs/
  2026/
    05 May/
      2026-05-14.md
```

## Template
Start from [[Templates/Daily Log Template]].

## How To Use
- Log the day quickly here.
- Link detailed sessions under the right category.
- Add JeFIT screenshots to [[Assets/JeFIT Screenshots/JeFIT Screenshots]] and link them from the strength section.
- Move bigger reflections into [[_Observations Hub]].
